IMPORTANT SECURITY LIMITATION
-----------------------------
This free version is a static Progressive Web App. The student access screen is a
basic deterrent, not the same as a private membership login or secure server-side
authentication. A technically skilled person could inspect or bypass client-side
code. For stronger protection, use individual Kajabi member logins or a backend
authentication service.

HOW TO PUBLISH FREE WITH GITHUB PAGES
-------------------------------------
1. Create a free GitHub account.
2. Create a new PUBLIC repository named:
   deba-principles-real-estate
3. Upload the CONTENTS of this folder.
   index.html must be in the repository root.
4. Open repository Settings > Pages.
5. Under Build and deployment:
   Source: Deploy from a branch
   Branch: main
   Folder: /(root)
6. Save and wait for GitHub to generate the public HTTPS link.

ANDROID INSTALLATION
--------------------
1. Open the public app link in Chrome.
2. Tap Install App in the app header.
3. Confirm installation.
If no prompt appears, open Chrome's menu and choose Add to Home Screen.

IPHONE / IPAD INSTALLATION
--------------------------
1. Open the public app link in Safari.
2. Tap Share.
3. Choose Add to Home Screen.
4. Tap Add.
